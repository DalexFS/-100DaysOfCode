Aprenderemos de HTML5

Las etiquetas HTML son las que indican al navegador como debe interpretar el contenido recogido entre dichas etiquetas. Estas etiquetas cuentan además con atributos que determinan como será el comportamiento específico que tendrá la etiqueta.

Etiquetas esenciales

Para el SEO y la estructura de nuestra pagina web

Estructura HTML

  • header: Tendra la cabecera de dicha pagina web
  • nav: Usadas para definir el contenido que será la sección de navegación de la web.
  • aside: Se encarga de contener datos extras, muy comun utilizarlas para paginas con publicidad.
  • main: Contenido de nuestra pagina web
  • h1,h2,h3,h4,h5,h6: se usan para explicar brevemente el contenido que irá a continuación.
  • article: Nos permite identificar un articulo en nuestro contenido web.
  • section: Nos permite insertar contenido generico en nuestro contenido web.
  • button: Nos permite insertar un boton en nuestro contenido web.
  • div: contenedor genérico.
  • footer: Es donde hubicamos nuestra información, como un contacto, nombre de la entidad, redes sociales etc...

Estructura Texto

  • cite: Indica el título de una obra
  • small: Utilizada para dejar un comentario aparte, del tipo una nota de derechos de autoría, u otros textos que no son esenciales para la comprensión del documento.
  • strong: Etiqueta para definir una palabra o conjunto de ellas como importantes.
  • a: Se utilizada para crear hiperenlaces en el documento HTML
  • p: Se utilizada para escribir párrafos de texto.
  • ol: Lista Ordenada.
  • ul: Lista Desordenada.
  • li: Elemento de la lista.

Estructura Contenido

  • img: Inserta una imagen en la página web
  • iframe: Es una etiqueta que sirve para anidar otro documento HTML dentro del documento principal.
  • object: Utilizada llamar a un recurso externo de la web. Este recurso será tratado como una imagen, o un recurso externo para ser procesado por un plugin
  • video: Se usa para reproducir video en la página web junto a sus archivos de audio.
  • audio: Usada para cargar en una web un archivo de audio o stream de audio.
  • svg: Se usa para llamar a una imagen vectorizada.

Modelo de Cajas

Una de los elementos mas importantes a la hora de desarrollar una pagina web, tenemos que entender lo que es el modelo de caja (se refiere a como HTML modela los elementos #etiquetas); con esto en mente podemos hacer manipulación del DOM (es una interfaz de programación para los documentos HTML y XML. Facilita una representación estructurada del documento y define de qué manera los programas pueden acceder, al fin de modificar, tanto su estructura, estilo y contenido).

Tenemos 3 propiedades importantes

  1. Bordes (Border)
  2. Margenes (Margin)
  3. Espacio Interno (padding)
  4. Contenido (Content)

Modelo Caja

Basandonos en este modelo, podemos ayudarnos con los estilos de CSS para ir moldeando una caja y su contenido, cabe afirmar que en ocaciones no tenemos una etiqueta para un contenido en especifico por esto HTML nos proporciona una etiqueta especial, es la etiqueta mas utilizada, nos sirve para seccionar contenidos en los modelos de caja, esta etiqueta es #div
